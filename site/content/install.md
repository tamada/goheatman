## :beer: Homebrew

```sh
$ brew tap tamada/brew
$ brew install heatman
```

## Go lang

```sh
$ go get github.com/tamada/goheatman
```

## :hammer_and_wrench: Build from source codes

```sh
$ git clone https://github.com/tamada/goheatman.git
$ cd goheatman
$ make
```
